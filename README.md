# CSV to Neo4j
Convert csv data to Neo4j Graph Database.
## How to use?
1. Set parameters in Config.py
2. Run "Python Csv2Neo.py + data_path" to convert csv data into Neo4j graph
## Dependencies
pandas, py2neo
